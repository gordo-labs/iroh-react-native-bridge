# Contributing

This repository is **open source** (MIT). Contributions welcome.

1. Open an issue describing the platform (iOS/Android) and Iroh version target.
2. Keep the API surface minimal — see [TASK.md](./TASK.md).
3. PRs should include device or CI evidence for native builds.

Upstream references: [n0-computer/iroh](https://github.com/n0-computer/iroh), [iroh-ffi](https://github.com/n0-computer/iroh-ffi) (patterns only).
